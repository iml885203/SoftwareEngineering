## 專案環境建置
### 1. Node.js & NPM
#### windows

* <https://nodejs.org/en/>下載安裝檔安裝
* 安裝完成後，開啟命令提示字元
* 輸入`node -v`和`npm -v`
* 有版本出現，代表安裝完成

### 2. Sails(Nodejs MVC框架)

#### windows
```
npm -g install sails
```
* 輸入`sails -v`
* 有版本出現，代表安裝完成




## 專案下載與建置
```
git clone https://github.com/iml885203/SoftwareEngineering.git
cd SoftwareEngineering
npm install
sails lift
```
* 執行`sails lift`後會開啟server
* 這開啟瀏覽器連到`http://localhost:1337/`就可以看到網頁
